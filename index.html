Introdução ao HTML

  • O que é ?

    voltar

    HTML, que significa HyperText Markup Language (Linguagem de Marcação de Hipertexto em português), é uma linguagem de marcação padrão usada para criar páginas web. Desenvolvido inicialmente por Tim Berners-Lee em 1991, o HTML desempenha um papel fundamental na estruturação e organização do conteúdo em documentos web.

  • História do HTML.

    voltar

    1. 1989 - A Proposta Inicial

      • O HTML foi proposto pela primeira vez por Tim Berners-Lee, um cientista da computação britânico, enquanto trabalhava no CERN (Organização Europeia para Pesquisa Nuclear).
      • Berners-Lee escreveu uma proposta chamada "Information Management: A Proposal", que delineava o conceito de um sistema de hipertexto para facilitar o compartilhamento de informações entre pesquisadores.

    2. 1991 - O Nascimento do HTML

      • Em 1991, Berners-Lee introduziu o HTML como parte do projeto WorldWideWeb (um navegador e editor) e do primeiro servidor web.
      • A versão inicial do HTML tinha tags básicas para estruturar documentos, incluindo cabeçalhos, parágrafos e links.

    3. 1995 - HTML 2.0

      • A primeira especificação formal do HTML foi lançada em 1995 como HTML 2.0 pelo IETF (Internet Engineering Task Force) e pela IAB (Internet Architecture Board).
      • Introduziu novos recursos como formulários e suporte para scripts.

    4. 1997 - HTML 3.2

      • Lançado em 1997, o HTML 3.2 trouxe mais recursos, incluindo tabelas e applets Java.
      • Este período também viu a introdução do CSS (Cascading Style Sheets) para separar a apresentação do conteúdo.

    5. 1999 - HTML 4.01

      • HTML 4.01, lançado em 1999, foi uma revisão significativa, introduzindo a noção de DTD (Document Type Definition) e suporte para documentos XHTML (HTML em XML).
      • Também foram adicionados recursos como frames e formulários avançados.

    6. 2008 - HTML5

      • HTML5, desenvolvido pelo grupo WHATWG (Web Hypertext Application Technology Working Group) e W3C (World Wide Web Consortium), começou a ganhar destaque em 2008.
      • HTML5 trouxe melhorias significativas, como elementos semânticos, API de áudio e vídeo, suporte para armazenamento local e canvas.

    7. 2020 - HTML Living Standard

      • O W3C abandonou a prática de versões numeradas e, a partir de 2019, adotou o conceito de "HTML Living Standard", indicando que HTML agora é uma especificação em evolução constante.

  • Como funciona uma página HTML básica.

    voltar

    Uma página HTML básica é composta por diferentes elementos que ajudam a estruturar e apresentar o conteúdo na web. Aqui está uma descrição sem comandos específicos

    1. Doctype Declaration: Declara o tipo de documento HTML que está sendo utilizado.
    2. Elemento HTML: Contém todo o conteúdo da página e possui duas seções principais: head e body.
    3. Elemento head: Contém informações sobre a página, como o título, links para folhas de estilo, scripts e outros metadados.
    4. Elemento body: Contém o conteúdo principal da página, como texto, imagens, links e outros elementos visíveis.
    5. Elementos de Conteúdo: Dentro do body, são incluídos vários elementos para criar o conteúdo desejado, como cabeçalhos, parágrafos, imagens, links, listas e formulários.
    6. Encerramento Adequado: Cada tag aberta deve ser fechada corretamente. Algumas tags, como imagens, podem ser autocontidas e não precisam de uma tag de fechamento separada.

Tags HTML Fundamentais

voltar

  • h1 até h6 -Cabeçalhos

    Define títulos de diferentes níveis. h1 é o mais importante, e h6 é o menos. Usado para estruturar hierarquicamente o conteúdo da página.

  • p - Parágrafo

    Define um parágrafo de texto. É usado para separar blocos de texto e proporcionar uma organização visual na página.

  • mark - Marcar Texto

    Define um trecho de texto que deve ser destacado ou marcado. Geralmente, o texto marcado é realçado com uma cor de fundo.

  • small - Texto Pequeno

    Reduz o tamanho do texto. Pode ser usado para notas de rodapé ou informações secundárias.

  • i - Texto Itálico:

    Indica que o texto deve ser exibido em itálico. Anteriormente, era frequentemente usado para ênfase, mas 'em' é mais apropriado semanticamente para ênfase.

  • u - Texto Sublinhado

    Indica que o texto deve ser sublinhado. No entanto, o uso do sublinhado é mais comum em links, e o estilo é geralmente controlado por CSS.

  • strong - Texto Forte/Negrito

    Indica que o texto deve ser enfatizado de maneira forte. O texto é geralmente renderizado em negrito.

  • ol, ul, li - Listas Ordenadas e Não Ordenadas

    • ol cria uma lista ordenada (numerada).
    • ul cria uma lista não ordenada (com marcadores).
    • li define um item de lista dentro de ol ou ul.

  • a - Âncora/Link

    Cria um link para outra página ou recurso. O atributo href especifica o destino do link.

  • hr - Linha Horizontal

    Cria uma linha horizontal para separar visualmente o conteúdo da página.

  • sub e sup - Texto Sobrescrito e Subscrito

    • sub é usado para texto subscrito (abaixo da linha).
    • sup é usado para texto sobrescrito (acima da linha).

  • blockquote - Bloco de Citação

    Usado para citar texto de outra fonte. O texto dentro desta tag é frequentemente indentado para indicar que é uma citação.
